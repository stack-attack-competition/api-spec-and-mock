## Description

This is an api specification, example implementation and mock data source for the Stack Attack #1 event.

## Installation

```bash
$ npm install
```

## Running the app

```bash
$ npm run start:dev
```

the api swagger spec will be on the `http://localhost:3000/api`

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
